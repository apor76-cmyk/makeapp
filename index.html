🔑 GitHub 토큰 설정

토큰이 설정되지 않았습니다.

파일 목록을 apor76-cmyk/data 저장소에 저장합니다.
토큰은 이 기기 브라우저에만 저장되며, 다른 기기에서도 각각 입력이 필요합니다.

① github.com/settings/tokens 접속
② Generate new token (classic) 클릭
③ repo 권한 체크 후 생성
④ 생성된 토큰을 아래에 붙여넣기

입력값은 이 브라우저의 localStorage에만 저장됩니다.

파일 추가

이 index.html과 같은 저장소에 있는 HTML 파일명을 입력하세요.

파일 제거

목록에서 을(를) 제거합니다.
실제 파일과 저장소의 파일은 삭제되지 않습니다.

📖 GitHub Pages 설정 방법

1
GitHub 저장소 만들기
github.com → New repository → Public 선택 → Create
2
파일 업로드
저장소 → Add file → Upload files → index.html 등 업로드
3
GitHub Pages 활성화
Settings → Pages → Branch: main / /(root) → Save
4
사이트 주소
https://사용자명.github.io/저장소명/
