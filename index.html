📖 GitHub Pages 설정 방법

1
GitHub 저장소 만들기
github.com → New repository → 이름 입력 → Public 선택 → Create
2
파일 업로드
저장소 페이지 → Add file → Upload files
index.html, 2028_admission_guide.html, 고교학점제_학부모_가이드.html 등 모두 업로드
3
GitHub Pages 활성화
저장소 → Settings → Pages → Source: Deploy from a branch
Branch: main / / (root) 선택 → Save
4
사이트 주소 확인
약 1~2분 후 Settings → Pages에서 주소 확인
형식: https://사용자명.github.io/저장소명/
5
새 파일 추가 시
HTML 파일을 저장소에 업로드 → index.html에서 ＋ 파일 추가 클릭 → 파일명 입력 → 저장
